**# Skills**




\- Nome: Análise de Identidade Visual

  Descrição: Auditar e validar aplicação da identidade visual em materiais e touchpoints

  Input:

    - Material para auditar (site, PDF, imagem, vídeo)

    - Guidelines da identidade visual

  Output:

    - Report de consistência com inconsistências listadas e correções sugeridas

  Tools:

    - Identidade Visual da Marca

    - Brand Audit Tool

  Processo:

    1. Carregar guidelines da identidade visual

    2. Analisar uso do logo (proporção, espaço, versão correta)

    3. Validar paleta de cores (tokens corret, contrastes)

    4. Validar tipografia (família, hierarquia, tamanhos)

    5. Verificar elementos visuais (ícones, imagens, padrões)

    6. Gerar report de consistência com score

  Regras:

    - Score mínimo de consistência: 90%

    - Logo nunca distorcido ou com fundo errado

    - Cores devem usar tokens do design system

    - Flaggar cada inconsistência com screenshot e correção




\- Nome: Análise de UI/UX

  Descrição: Avaliar interfaces sob critérios de usabilidade, acessibilidade e experiência

  Input:

    - Interface para analisar (site, app, plataforma)

    - Público-alvo e contexto de uso

  Output:

    - Report de UI/UX com problemas, severidade e recomendações

  Tools:

    - UI/UX Analyzer

    - Design System

  Processo:

    1. Mapear fluxos principais do usuário

    2. Aplicar heurísticas de Nielsen (10 critérios)

    3. Testar acessibilidade WCAG (A, AA, AAA)

    4. Avaliar consistência com design system

    5. Medir carga cognitiva e complexidade visual

    6. Calcular score de usabilidade

    7. Listar problemas por severidade (crítico, alto, médio, baixo)

    8. Gerar recomendações priorizadas

  Regras:

    - Problemas críticos devem ser resolvidos em 48h

    - Acessibilidade AA é mínimo obrigatório

    - Avaliar mobile-first

    - Incluir screenshots de cada problema




\- Nome: Análise de Branding Competitivo

  Descrição: Analisar branding de concorrentes e mercado com base em dados

  Input:

    - Lista de concorrentes

    - Canais para analisar (site, redes, materiais, campanhas)

  Output:

    - Report competitivo com análise visual, posicionamento e oportunidades

  Tools:

    - Market Research Tool

    - Competitive Brand Tracker

  Processo:

    1. Identificar concorrentes diretos e indiretos

    2. Coletar dados visuais de cada concorrente (logo, cores, tipografia, tom)

    3. Analisar posicionamento e promessa de marca

    4. Avaliar presença digital (site, redes, conteúdo)

    5. Identificar tendências de branding no setor

    6. Comparar Neural Trax vs concorrentes em critérios chave

    7. Identificar gaps e oportunidades

    8. Gerar report com recomendações estratégicas

  Regras:

    - Benchmark é análise, não cópia

    - Incluir dados visuais (screenshots, exemplos)

    - Focar em diferenciação, não em seguir tendência

    - Priorizar recomendações por impacto




\- Nome: Evolução de Design System

  Descrição: Propor evoluções no design system baseadas em dados e necessidades

  Input:

    - Dados de uso do design system (componentes mais usados, gaps)

    - Feedback de designers e devs

    - Novas necessidades de produto

  Output:

    - Proposta de evolução com justificativa e especificação

  Tools:

    - Design System

    - UI/UX Analyzer

  Processo:

    1. Analisar dados de uso atual do design system

    2. Identificar componentes faltando ou problemáticos

    3. Coletar feedback de usuários internos

    4. Pesquisar padrões de mercado

    5. Propor novos componentes ou alterações

    6. Especificar mudanças (tokens, variantes, estados)

    7. Validar impacto em acessibilidade e consistência

    8. Gerar proposta para aprovação

  Regras:

    - Breaking changes exigem migração plan

    - Novos componentes devem servir 2+ casos de uso

    - Manter backward compatibility quando possível

    - Documentar changelog




\- Nome: Diretrizes de Comunicação da Marca

  Descrição: Criar e atualizar guidelines de comunicação visual e verbal

  Input:

    - Posicionamento atual da marca

    - Público-alvo e persona

    - Tom de voz desejado

  Output:

    - Guia de comunicação atualizado (tom, voz, linguagem, visual)

  Tools:

    - Guia de Comunicação da Marca

    - Identidade Visual da Marca

  Processo:

    1. Revisar posicionamento e proposta de valor

    2. Definir tom de voz por contexto (formal, casual, técnico)

    3. Criar guidelines de copywriting (estrutura, linguagem, CTAs)

    4. Definir regras de uso de imagens e ilustrações

    5. Estabelecer do's and don'ts de comunicação

    6. Gerar guia estruturado

    7. Validar com Head de Marketing

  Regras:

    - Tom consistente em todos os canais

    - Linguagem acessível e inclusiva

    - Exemplos práticos de certo vs errado

    - Atualizar trimestralmente




\- Nome: Monitoramento de Tendências

  Descrição: Monitorar tendências de design, branding e comunicação

  Input:

    - Setor e contexto de mercado

    - Período de análise

  Output:

    - Report de tendências com relevância para Neural Trax

  Tools:

    - Market Research Tool

    - Competitive Brand Tracker

  Processo:

    1. Pesquisar tendências de design e branding

    2. Filtrar por relevância ao setor

    3. Analisar adoção por líderes de mercado

    4. Avaliar impacto potencial na Neural Trax

    5. Recomendar adoção ou não com justificativa

    6. Gerar report de tendências

  Regras:

    - Tendência ≠ obrigação: avaliar fit com marca

    - Priorizar tendências com dados de adoção

    - Incluir exemplos reais de aplicação

    - Recomendar timing de adoção
