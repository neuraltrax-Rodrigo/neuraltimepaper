**# Tools**




\- Nome: Agenteda (Google Agent)

  Tipo: API

  Função: Buscar e extrair leads de empresas no Google via Agenteda

  Permissões:

  - Buscar empresas por segmento e localização

  - Extrair nome, telefone (WhatsApp), email, site

  - Identificar decisor (CEO, dono, gerente)




\- Nome: Asana CRM

  Tipo: Integração

  Função: Cadastrar e gerenciar leads como clientes no Asana

  Permissões:

  - Criar novo cliente com dados completos

  - Atualizar status de prospecção

  - Registrar interações WhatsApp

  - Mover lead por etapas do funil

  - Agendar reuniões no calendário




\- Nome: WhatsApp Sender

  Tipo: API

  Função: Enviar mensagens de prospecção e follow-up via WhatsApp

  Permissões:

  - Enviar mensagens personalizadas (máx 2-3 linhas)

  - Agendar envios em horário comercial

  - Rastrear entregas e respostas

  - Responder automaticamente com base no roteiro




\- Nome: WhatsApp Template Manager

  Tipo: Script

  Função: Gerenciar templates do roteiro (Etapas 1-6.4 + 4 follow-ups)

  Permissões:

  - Template de saudação (Etapa 1)

  - Template de identificação (Etapas 2-3)

  - Template de oferta de agendamento (Etapa 5)

  - Templates de follow-up (1º ao 4º)

  - Personalização com variáveis (nome, empresa, dor)




\- Nome: Lead Qualifier BANT

  Tipo: Script

  Função: Qualificar leads com framework BANT simplificado

  Permissões:

  - Avaliar Budget (se perguntar preço \= sinal)

  - Avaliar Authority (se é decisor)

  - Avaliar Need (dor expressa)

  - Avaliar Timeline (urgência)

  - Classificar: quente (agenda), morno (nurturing), frio (descarta)




\- Nome: Agendamento

  Tipo: Integração

  Função: Consultar horários vagos e confirmar agendamento

  Permissões:

  - Consultar disponibilidade no calendário

  - Agrupar horários por manhã/tarde

  - Confirmar agendamento com cliente

  - Enviar lembrete 1h antes
