# Tools

## Google Sheets (via Zapier MCP)

* **Tipo:** Integração / MCP Server
* **Função:** Base de dados financeira principal — fluxo de caixa, DRE, precificação e projeções
* **Permissões:**
  * Ler e escrever em planilhas financeiras
  * Criar abas e formatar relatórios automaticamente
  * Buscar linhas por cliente, projeto ou período

## Gmail (via Gmail MCP)

* **Tipo:** Integração / MCP Server
* **Função:** Comunicação financeira com clientes — envio de cobranças, recibos e alertas de vencimento
* **Permissões:**
  * Enviar e-mails financeiros a partir de endereço configurado
  * Ler threads de negociação financeira com clientes
  * Arquivar comprovantes e notas fiscais recebidas

## Slack MCP

* **Tipo:** Integração / MCP Server
* **Função:** Notificações financeiras internas para o CEO e time
* **Permissões:**
  * Enviar alertas de caixa, inadimplência e vencimentos
  * Publicar relatórios financeiros resumidos em canal dedicado
  * Receber e processar solicitações financeiras via mensagem

## Claude API (Anthropic)

* **Tipo:** API
* **Função:** Análise financeira, geração de relatórios narrativos e suporte a decisões com base em dados
* **Permissões:**
  * Processar dados financeiros e gerar insights em linguagem natural
  * Elaborar projeções e cenários com base em inputs estruturados
  * Redigir comunicações financeiras para clientes

## Zapier MCP

* **Tipo:** Integração / MCP Server
* **Função:** Automações financeiras — gatilhos de cobrança, sincronização de dados e alertas cross-platform
* **Permissões:**
  * Disparar zaps a partir de eventos financeiros (vencimento, pagamento, novo contrato)
  * Sincronizar dados entre Google Sheets e outros sistemas
  * Integrar com plataformas de emissão de NF-e

## Sistema de Emissão de NF-e

* **Tipo:** API / Integração
* **Função:** Emissão e arquivamento de notas fiscais de serviço
* **Permissões:**
  * Emitir NFS-e por cliente e projeto
  * Consultar status de notas emitidas
  * Baixar XMLs e PDFs para arquivamento
