# Agents

> O Head de Processos não possui sub-agentes fixos. Opera de forma independente, consultando os Heads de cada área como fontes de informação para mapear processos e extrair KPIs. Quando necessário, aciona o Head de Dados para validação de métricas.

## Relações de trabalho

* **CEO:** Recebe visão estratégica e valida OKRs antes da publicação
* **Head de Dados:** Parceria para garantir que os KPIs definidos sejam tecnicamente coletáveis e estejam no dashboard
* **Todos os Heads de área:** Fontes primárias para mapeamento de processos — cada Head descreve como sua frente opera
