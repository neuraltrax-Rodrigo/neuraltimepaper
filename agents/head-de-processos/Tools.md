# Tools

## Claude API (Anthropic)

* **Tipo:** API
* **Função:** Redação de documentação de processos, estruturação de OKRs e análise de KPIs
* **Permissões:**
  * Gerar SOPs, fluxogramas em texto e documentação estruturada
  * Sugerir KPIs por frente com base em benchmarks de mercado
  * Redigir OKRs no formato padrão (Objetivo + Key Results mensuráveis)

## Google Sheets (via Zapier MCP)

* **Tipo:** Integração
* **Função:** Repositório de KPIs definidos, OKRs do período e registro de processos mapeados
* **Permissões:**
  * Criar e manter planilha mestre de KPIs por área
  * Registrar OKRs ativos com metas, responsável e prazo
  * Acompanhar status de documentação por processo

## Slack MCP

* **Tipo:** Integração / MCP Server
* **Função:** Coleta de informações com os Heads e entrega de documentação ao CEO
* **Permissões:**
  * Enviar perguntas de mapeamento aos Heads de área
  * Publicar novos processos documentados e OKRs no canal executivo
  * Receber feedback e validações via thread

## Asana MCP

* **Tipo:** Integração / MCP Server
* **Função:** Gestão dos processos de documentação em andamento e registro de OKRs
* **Permissões:**
  * Criar tarefas de mapeamento por área com prazo e responsável
  * Registrar OKRs como projetos com Key Results como tarefas mensuráveis
  * Acompanhar backlog de processos pendentes de documentação
