# Tools

## Claude API (Anthropic)

* **Tipo:** API
* **Função:** Motor de raciocínio estratégico do CEO — análises, sínteses, decisões e comunicações executivas
* **Permissões:**
  * Processar contexto de múltiplas frentes e gerar diretrizes estratégicas
  * Redigir comunicações executivas para clientes, parceiros e time
  * Sintetizar relatórios das lideranças em visão unificada

## Slack MCP

* **Tipo:** Integração / MCP Server
* **Função:** Canal principal de comunicação com todas as lideranças e o fundador
* **Permissões:**
  * Enviar e receber mensagens em todos os canais relevantes
  * Publicar briefings, decisões e alertas executivos
  * Monitorar canais de cada frente para visibilidade operacional

## Asana MCP

* **Tipo:** Integração / MCP Server
* **Função:** Gestão de projetos, OKRs, roadmap e tarefas das lideranças
* **Permissões:**
  * Criar, atribuir e acompanhar tarefas e projetos
  * Visualizar status de todas as frentes em tempo real
  * Escalar bloqueios e repriorizar iniciativas

## Google Calendar MCP

* **Tipo:** Integração / MCP Server
* **Função:** Gestão de agenda executiva — reuniões, reviews e rituais de liderança
* **Permissões:**
  * Criar e atualizar eventos de reunião com lideranças
  * Agendar rituais recorrentes (weekly, monthly review, OKR check-in)
  * Verificar disponibilidade para tomada de decisão em tempo real

## Gmail MCP

* **Tipo:** Integração / MCP Server
* **Função:** Comunicação formal com clientes, parceiros e stakeholders externos
* **Permissões:**
  * Ler e responder e-mails de alta prioridade
  * Redigir e enviar comunicações executivas
  * Arquivar e categorizar correspondências estratégicas

## Google Sheets (via Zapier MCP)

* **Tipo:** Integração
* **Função:** Dashboard executivo consolidado — KPIs, OKRs, financeiro e performance por frente
* **Permissões:**
  * Ler dados de todas as frentes consolidados
  * Atualizar status de OKRs e iniciativas estratégicas
  * Gerar visões comparativas de performance
