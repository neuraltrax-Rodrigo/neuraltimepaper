# Tools

## Claude API (Anthropic)

* **Tipo:** API
* **Função:** Análise de funil, diagnóstico de gargalos, síntese de dados do squad e apoio à tomada de decisão tática
* **Permissões:**
  * Analisar dados de pipeline e identificar padrões de perda por etapa
  * Sintetizar relatórios do SDR, BDR, Closer e Transcrição em visão consolidada
  * Gerar recomendações táticas de desbloqueio de deals parados

## Google Sheets (via Zapier MCP)

* **Tipo:** Integração
* **Função:** Dashboard do pipeline de vendas — controle de leads, reuniões, propostas e fechamentos
* **Permissões:**
  * Ler e atualizar pipeline com dados do SDR, BDR e Closer
  * Calcular taxas de conversão por etapa e por agente
  * Acompanhar metas semanais e mensais de MRR novo

## Slack MCP

* **Tipo:** Integração / MCP Server
* **Função:** Comunicação operacional com o squad e reporte ao Head de Growth
* **Permissões:**
  * Receber relatórios diários do SDR, BDR, Closer e Transcrição
  * Enviar orientações táticas e ajustes de abordagem ao squad
  * Reportar performance semanal ao Head de Growth
  * Alertar sobre deals críticos parados ou perdas relevantes

## Asana MCP

* **Tipo:** Integração / MCP Server
* **Função:** Gestão de tarefas do squad e acompanhamento de negociações críticas
* **Permissões:**
  * Criar tarefas de follow-up para deals parados
  * Acompanhar contratos em processo de assinatura
  * Registrar ações de desbloqueio e seus resultados

## Zapier MCP

* **Tipo:** Integração / MCP Server
* **Função:** Automações de funil — alertas de lead novo, atualização de pipeline e notificações entre plataformas
* **Permissões:**
  * Disparar alerta automático quando lead entra no funil
  * Notificar Closer quando SDR ou BDR agenda reunião qualificada
  * Sincronizar dados de pipeline com Google Sheets
