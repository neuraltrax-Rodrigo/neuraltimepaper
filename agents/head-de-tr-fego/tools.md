# Tools

## Claude API (Anthropic)

* **Tipo:** API
* **Função:** Análise de performance, diagnóstico de campanhas e geração de recomendações táticas
* **Permissões:**
  * Interpretar dados de tráfego e gerar diagnóstico de canal com recomendações
  * Redigir briefings de campanha e orientações para o squad
  * Sintetizar relatório semanal consolidado para o Head de Growth e CEO

## Google Sheets (via Zapier MCP)

* **Tipo:** Integração
* **Função:** Dashboard de performance de tráfego — CPL, CAC, leads por canal, budget e ROAS
* **Permissões:**
  * Consolidar dados de todas as fontes de tráfego
  * Calcular CPL e qualidade de lead por canal semanalmente
  * Acompanhar budget gasto vs. planejado por campanha

## Slack MCP

* **Tipo:** Integração / MCP Server
* **Função:** Comunicação com o squad e reporte ao Head de Growth e CEO
* **Permissões:**
  * Receber relatórios diários e semanais do squad
  * Enviar orientações táticas e ajustes de campanha
  * Publicar relatório semanal de tráfego

## Zapier MCP

* **Tipo:** Integração / MCP Server
* **Função:** Automações de coleta de dados e alertas de performance entre plataformas
* **Permissões:**
  * Sincronizar dados de Meta Ads e Google Ads com o Google Sheets
  * Disparar alerta automático quando CPL ultrapassar threshold
  * Acionar notificação quando campanha for pausada por orçamento esgotado

## Google Analytics / Meta Ads / Google Ads

* **Tipo:** API / Integração
* **Função:** Fontes primárias de dados de performance de tráfego
* **Permissões:**
  * Ler métricas de campanhas: impressões, cliques, leads, CPL, ROAS, conversões
  * Exportar relatórios por período e canal
  * Monitorar anomalias de performance em tempo real
