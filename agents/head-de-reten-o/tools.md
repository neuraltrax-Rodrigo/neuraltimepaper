# Tools

## Claude API (Anthropic)

* **Tipo:** API
* **Função:** Análise de saúde de clientes, síntese de feedback e geração de planos de intervenção
* **Permissões:**
  * Identificar padrões de risco de churn com base em dados de uso e NPS
  * Redigir comunicações personalizadas de retenção e upsell
  * Sintetizar relatório mensal de saúde da base

## Google Sheets (via Zapier MCP)

* **Tipo:** Integração
* **Função:** Base de dados da carteira — health score, NPS, tickets, contratos e oportunidades de upsell
* **Permissões:**
  * Manter e atualizar health score de cada cliente ativo
  * Registrar interações de CS, NPS e suporte por conta
  * Acompanhar pipeline de upsell e expansões fechadas

## Slack MCP

* **Tipo:** Integração / MCP Server
* **Função:** Comunicação interna com o squad e alertas críticos ao CEO
* **Permissões:**
  * Receber relatórios do squad (Customer Experience, NPS, Suporte, Churn Recovery, Upsell)
  * Enviar alertas de risco de churn ao CEO
  * Publicar relatório semanal de retenção ao CEO

## Gmail MCP

* **Tipo:** Integração / MCP Server
* **Função:** Comunicação formal com clientes — check-ins, pesquisas de NPS e abordagens de retenção/upsell
* **Permissões:**
  * Enviar e-mails de relacionamento e recuperação para clientes
  * Disparar pesquisas de NPS automatizadas
  * Acompanhar respostas de clientes em risco

## Asana MCP

* **Tipo:** Integração / MCP Server
* **Função:** Gestão de ações de retenção, planos de recuperação e oportunidades de upsell
* **Permissões:**
  * Criar tarefas de intervenção por cliente em risco
  * Acompanhar planos de churn recovery em andamento
  * Registrar oportunidades de upsell identificadas pelo Upsell Agent
