**# Tools**



\- Nome: Meta Ads Manager API

  Tipo: Integração

  Função: Gerenciar campanhas no Facebook e Instagram Ads

  Permissões:

    - Criar, editar e pausar campanhas

    - Acessar dados de performance em tempo real

    - Gerenciar públicos e audiences

    - Upload de criativos


\- Nome: Google Ads API

  Tipo: Integração

  Função: Gerenciar campanhas no Google Ads

  Permissões:

    - Criar e gerenciar campanhas Search, Display, YouTube

    - Acessar keywords e search terms

    - Modificar bids e orçamentos

    - Implementar negative keywords


\- Nome: TikTok Ads Manager

  Tipo: Integração

  Função: Executar campanhas no TikTok

  Permissões:

    - Criar campanhas e ad groups

    - Upload de vídeos

    - Segmentação por interesse e comportamento

    - Acesso a métricas de engagement


\- Nome: LinkedIn Campaign Manager

  Tipo: Integração

  Função: Gerenciar anúncios B2B no LinkedIn

  Permissões:

    - Targeting por cargo e empresa

    - Lead Gen Forms

    - Matched Audiences

    - Reporte de conversões B2B


\- Nome: Agente de KPIs

  Tipo: Integração

  Função: Receber dados consolidados de performance

  Permissões:

    - Leitura de ROAS, CPA, CTR, conversões

    - Dados históricos de campanhas

    - Benchmarks por canal e indústria

    - Alertas de performance anômala


\- Nome: Agente Alfredo Soares

  Tipo: Integração

  Função: Receber insights de estratégia e público

  Permissões:

    - Estratégia de marketing vigente

    - Persona e dados de audiência

    - Posicionamento de marca

    - Calendário de lançamentos


\- Nome: Copywriter

  Tipo: Integração

  Função: Receber copies aprovadas para campanhas

  Permissões:

    - Copies de anúncios validadas

    - Variações para A/B testing

    - Diretrizes de messaging


\- Nome: Branding Guidelines

  Tipo: Integração

  Função: Validar criativos contra diretrizes de marca

  Permissões:

    - Cores, fontes, elementos visuais

    - Tom e voz da marca

    - Do's and Don'ts de comunicação


\- Nome: Google Analytics 4

  Tipo: Integração

  Função: Acessar dados de comportamento do usuário

  Permissões:

    - Eventos de conversão

    - Journeys de usuário

    - Attribution data

    - Audience insights


\- Nome: Google Tag Manager

  Tipo: Script

  Função: Gerenciar tags e pixels no site

  Permissões:

    - Instalar e configurar tags

    - Criar triggers de evento

    - Debug de implementação

    - Versionamento de containers


\- Nome: Pixel Helper Suite

  Tipo: Script

  Função: Validar instalação de pixels

  Permissões:

    - Meta Pixel Helper

    - TikTok Pixel Helper

    - Google Tag Assistant

    - LinkedIn Insight Tag checker


\- Nome: A/B Test Generator

  Tipo: Script

  Função: Planejar e calcular testes A/B

  Permissões:

    - Calcular sample size necessário

    - Determinar significância estatística

    - Sugerir duração de teste

    - Analisar resultados


\- Nome: Budget Allocator

  Tipo: Script

  Função: Otimizar alocação de budget entre campanhas

  Permissões:

    - Analisar ROAS por campanha

    - Sugerir realocação de budget

    - Projetar impacto de mudanças

    - Controlar limites de gasto


\- Nome: CRM Integration

  Tipo: Integração

  Função: Sincronizar leads e clientes com plataformas de ads

  Permissões:

    - Upload de listas de clientes

    - Download de leads capturados

    - Sync de status de lead

    - Closed-loop attribution
