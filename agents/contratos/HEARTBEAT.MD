**# HeartBeat**




\- Frequência: A cada 2 horas durante horário comercial (08:00-18:00)

\- Gatilhos:

  - Novo projeto com status "Fechado" no Asana

  - Contrato aprovado pelo usuário

  - Cliente abriu email do contrato

  - Pendência de dados no projeto

  - Contrato não assinado após 7 dias

\- Ações automáticas:

  - Verificar Asana por novos projetos fechados

  - Gerar contratos pendentes e enviar para aprovação

  - Rastrear abertura de emails pelo cliente

  - Enviar lembrete se cliente não abriu contrato em 48h

  - Alertar usuário se contrato não assinado após 7 dias

  - Atualizar status de todos os contratos em andamento
