# Tools

## Claude API (Anthropic)

* **Tipo:** API
* **Função:** Desenvolvimento de estratégias de growth, análise de funil, criação de playbooks comerciais e síntese de insights de conversão
* **Permissões:**
  * Desenvolver e documentar estratégias de crescimento e go-to-market
  * Analisar dados de funil e gerar diagnóstico de gargalos com recomendações
  * Criar e iterar playbooks de vendas, scripts de abordagem e cadências
  * Simular perspectiva do Alfredo Soares em desafios estratégicos de growth

## Slack MCP

* **Tipo:** Integração / MCP Server
* **Função:** Comunicação com o CEO, Head de Dados, Head de Conversão e todo o squad de growth
* **Permissões:**
  * Enviar e receber mensagens em canais de growth, conversão e executivo
  * Publicar briefings de estratégia e resultados semanais para o squad
  * Receber alertas do Head de Dados sobre métricas de funil
  * Comunicar ajustes de estratégia ao time em tempo real

## Asana MCP

* **Tipo:** Integração / MCP Server
* **Função:** Gestão de iniciativas de growth, campanhas, sprints comerciais e OKRs da frente
* **Permissões:**
  * Criar e gerenciar projetos de growth e campanhas de aquisição
  * Atribuir tarefas ao Head de Conversão, SDR, BDR e demais agentes do squad
  * Monitorar progresso de iniciativas e identificar bloqueios
  * Alinhar OKRs de growth com o painel executivo do CEO

## Google Sheets (via Zapier MCP)

* **Tipo:** Integração
* **Função:** Dashboard de funil de vendas — pipeline, taxas de conversão por etapa, CAC, MRR novo e metas comerciais
* **Permissões:**
  * Ler e atualizar pipeline de vendas e métricas de conversão
  * Acompanhar MRR novo gerado pela frente de growth por semana/mês
  * Consolidar dados de SDR e BDR (leads contactados, reuniões agendadas, propostas enviadas, contratos fechados)
  * Calcular CAC e comparar com LTV para avaliar eficiência de aquisição

## Gmail MCP

* **Tipo:** Integração / MCP Server
* **Função:** Comunicação formal com prospects e clientes em etapas avançadas do funil
* **Permissões:**
  * Ler threads de negociação com prospects para contexto estratégico
  * Enviar comunicações de alto nível para contas estratégicas
  * Acompanhar propostas enviadas e status de resposta

## Zapier MCP

* **Tipo:** Integração / MCP Server
* **Função:** Automações de funil — notificações de lead novo, atualização de pipeline e gatilhos entre plataformas
* **Permissões:**
  * Disparar notificações automáticas quando lead entra no funil
  * Sincronizar dados de funil entre plataformas
  * Acionar cadências de follow-up automático via integrações
