# Tools

## Claude API (Anthropic)

* **Tipo:** API
* **Função:** Síntese de feedback de clientes, redação de PRDs, análise de priorização e geração de documentação de produto
* **Permissões:**
  * Redigir Product Requirements Documents (PRDs) e user stories
  * Sintetizar feedback de múltiplas fontes em insights acionáveis
  * Apoiar decisões de priorização com análise de impacto × esforço

## Asana MCP

* **Tipo:** Integração / MCP Server
* **Função:** Gestão do roadmap, backlog de features e épicos de produto
* **Permissões:**
  * Criar e priorizar épicos, features e tarefas no backlog
  * Atribuir itens ao CTO para execução técnica
  * Acompanhar progresso de entrega do roadmap

## Slack MCP

* **Tipo:** Integração / MCP Server
* **Função:** Coleta de inputs de produto com Heads e comunicação de decisões
* **Permissões:**
  * Receber feedback de produto do Head de Retenção, Head of Sales e CEO
  * Comunicar atualizações de roadmap e decisões de priorização
  * Criar canais de discussão de feature quando necessário

## Google Sheets (via Zapier MCP)

* **Tipo:** Integração
* **Função:** Matriz de priorização de features e registro de feedback de clientes
* **Permissões:**
  * Manter matriz de priorização (impacto × esforço × alinhamento estratégico)
  * Registrar e categorizar feedback de clientes por tema
  * Acompanhar métricas de adoção de features entregues

## Gmail MCP

* **Tipo:** Integração / MCP Server
* **Função:** Comunicação formal de roadmap e coleta de feedback estruturado com clientes estratégicos
* **Permissões:**
  * Enviar atualizações de produto para clientes selecionados
  * Receber e processar feedback estruturado de clientes
