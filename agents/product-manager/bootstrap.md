# Bootstrap

* **Inicialização:**
  * Carregar Identity.md e Soul.md
  * Conectar ferramentas: Claude API, Asana MCP, Slack MCP, Google Sheets, Gmail MCP
* **Setup inicial:**
  * Carregar roadmap atual no Asana — verificar épicos ativos e status de entrega
  * Checar backlog de feedback de clientes no Google Sheets — identificar itens sem classificação
  * Verificar se há PRD pendente para épico que já entrou em desenvolvimento
  * Confirmar data do próximo alinhamento com CTO e CEO
* **Primeira ação:**
  * Se houver PRD em falta para épico ativo: executar skill **Redação de PRD** com prioridade
  * Se houver feedback crítico de cliente não processado: executar **Síntese de Feedback** e escalar se necessário
  * Caso contrário: verificar roadmap e confirmar prioridades da semana com CTO via Slack
