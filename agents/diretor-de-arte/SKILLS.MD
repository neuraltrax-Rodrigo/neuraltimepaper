**# Skills**




\- Nome: Conceito Visual de Campanha

  Descrição: Definir direção visual completa para cada campanha (mood, estilo, paleta, referências)

  Input:

    - Briefing da campanha (objetivo, público, mensagem)

    - Copy da campanha (do Copywriter)

    - Diretrizes de marca (do Branding)

  Output:

    - Conceito visual documentado com mood, paleta, referências, estilo fotográfico, tipografia de destaque

  Tools:

    - Visual Trend Board

    - Color Palette Generator

    - Branding Guidelines

    - Copywriter

  Processo:

    1. Analisar briefing e copy da campanha

    2. Identificar emoção/mensagem visual a transmitir

    3. Pesquisar referências visuais alinhadas com conceito

    4. Definir mood board (3-5 referências visuais)

    5. Selecionar/gerar paleta de cores da campanha

    6. Definir estilo fotográfico (lifestyle, produto, abstrato, minimal)

    7. Especificar tipografia de destaque (se aplicável)

    8. Documentar conceito visual completo

  Regras:

    - Conceito deve apoiar a copy, não competir

    - Paleta deve ser harmônica e acessível (contraste WCAG AA)

    - Referências devem ser claras e acionáveis para designers

    - Dentro das guidelines de marca (variação permitida, não violação)




\- Nome: Composição Visual de Peças

  Descrição: Orientar composição de peças visuais (layout, hierarquia, balanço, espaço)

  Input:

    - Tipo de peça (ad, LP, post, vídeo)

    - Copy e elementos obrigatórios (logo, CTA)

    - Conceito visual da campanha

  Output:

    - Direção de composição com layout, hierarquia, posicionamento de elementos

  Tools:

    - Composition Analyzer

    - Branding Guidelines

  Processo:

    1. Definir formato e dimensões da peça

    2. Estabelecer hierarquia visual (o que olho vê primeiro, segundo, terceiro)

    3. Definir layout grid e zonas de atenção

    4. Posicionar copy conforme hierarquia (headline > body > CTA)

    5. Posicionar elementos visuais (imagem, logo, badge)

    6. Verificar espaço vazio e respiro visual

    7. Validar composição com Composition Analyzer

    8. Gerar guia de composição para designer

  Regras:

    - CTA sempre visível e claro

    - Logo posicionado consistentemente

    - Espaço vazio intencional, não acidental

    - Contraste suficiente entre texto e fundo

    - Mobile-first: testar em formato pequeno




\- Nome: Curadoria Visual

  Descrição: Buscar, selecionar e organizar imagens e referências visuais

  Input:

    - Conceito visual da campanha

    - Estilo e mood definidos

    - Tipo de imagem necessária (lifestyle, produto, abstrato)

  Output:

    - Mood board curado com 5-10 imagens + referências de estilo

  Tools:

    - Image Curator Tool

    - Visual Trend Board

  Processo:

    1. Traduzir conceito visual em termos de busca

    2. Buscar imagens em bancos (Unsplash, Shutterstock, Pexels)

    3. Filtrar por qualidade, licença e alinhamento com conceito

    4. Selecionar 5-10 imagens principais

    5. Organizar por uso (hero, apoio, background, social)

    6. Adicionar notas de curadoria (por que essa imagem funciona)

    7. Gerar mood board documentado

  Regras:

    - Apenas imagens com licença comercial

    - Diversidade representativa quando aplicável

    - Cada imagem deve ter razão de ser no conceito

    - Incluir alternativas para cada seleção

    - Documentar fonte e créditos




\- Nome: Validação de Peças Visuais

  Descrição: Validar peças visuais produzidas contra conceito e guidelines

  Input:

    - Peça visual final (imagem, vídeo, LP)

    - Conceito visual da campanha

    - Guidelines de marca

  Output:

    - Report de validação com aprovações e ajustes necessários

  Tools:

    - Composition Analyzer

    - Branding Guidelines

  Processo:

    1. Verificar alinhamento com conceito visual (mood, estilo, paleta)

    2. Validar contra guidelines de marca (cores, tipografia, logo)

    3. Analisar composição (hierarquia, contraste, espaço, balanço)

    4. Verificar legibilidade de texto em todos os tamanhos

    5. Testar contraste para acessibilidade WCAG AA

    6. Avaliar impacto visual geral (funciona em 1 segundo?)

    7. Gerar report: aprovado, aprovado com ajustes, ou reprovado

  Regras:

    - Reprovado se violar guideline de marca

    - Aprovado com ajustes se problemas menores de composição

    - Aprovado se alinhado com conceito e dentro de guidelines

    - Sempre incluir sugestões concretas de ajuste




\- Nome: Direção Visual para Vídeos

  Descrição: Definir direção visual de vídeos (estilo, enquadramento, cores, motion)

  Input:

    - Script do vídeo (do Copywriter)

    - Objetivo e plataforma

    - Conceito visual da campanha

  Output:

    - Briefing visual do vídeo com estilo, enquadramento, paleta, motion graphics

  Tools:

    - Visual Trend Board

    - Color Palette Generator

    - Composition Analyzer

  Processo:

    1. Analisar script e identificar momentos visuais chave

    2. Definir estilo visual (live action, motion, mixed, screencast)

    3. Especificar enquadramento e composição por cena

    4. Definir paleta de cores do vídeo

    5. Especificar motion graphics e transições

    6. Definir tipografia on-screen (se aplicável)

    7. Gerar briefing visual completo para editor

  Regras:

    - Primeiros 3 segundos devem ser visualmente impactantes

    - Tipografia on-screen legível em mobile

    - Transições consistentes com mood da campanha

    - Motion graphics com propósito, não decorativo

    - Incluir referências visuais de estilo




\- Nome: Evolução Visual Baseada em Dados

  Descrição: Propor evoluções visuais baseadas em performance e tendências

  Input:

    - Dados de performance visual (CTR, engajamento por tipo de visual)

    - Tendências visuais do setor

    - Feedback de audiência

  Output:

    - Proposta de evolução visual com justificativa e especificação

  Tools:

    - Agente de KPIs

    - Visual Trend Board

    - Composition Analyzer

  Processo:

    1. Analisar performance por tipo de visual (foto, ilustração, motion, cor)

    2. Identificar patterns de alta performance

    3. Pesquisar tendências visuais relevantes

    4. Formular hipótese de evolução (ex: "motion aumenta CTR em X%")

    5. Especificar mudança proposta

    6. Validar contra identidade de marca

    7. Gerar proposta para aprovação do Head de Marketing

  Regras:

    - Evolução, não revolução: manter reconhecimento

    - Basear em dados, não preferência pessoal

    - Testar mudança antes de implementar totalmente

    - Documentar learning de cada evolução




\- Nome: Coordenação Copy + Visual

  Descrição: Unificar copy e conceito visual para máxima comunicação

  Input:

    - Copy da campanha (do Copywriter)

    - Conceito visual proposto

    - Objetivo de comunicação

  Output:

    - Direção criativa unificada com copy + visual integrados

  Tools:

    - Copywriter

    - Composition Analyzer

    - Branding Guidelines

  Processo:

    1. Analisar copy e identificar mensagem principal

    2. Definir como visual amplifica (não repete) a copy

    3. Estabelecer hierarquia: copy-first ou visual-first

    4. Definir pontos de integração (texto sobre imagem, cor que reforça emoção)

    5. Validar unidade de comunicação (mensagem clara em 3 segundos?)

    6. Gerar direção criativa completa

  Regras:

    - Copy e visual devem se complementar, não competir

    - Se um funciona sozinho, o outro está fraco

    - Testar em 3 segundos: a mensagem é clara?

    - Hierarquia definida pelo objetivo (emoção vs informação)
