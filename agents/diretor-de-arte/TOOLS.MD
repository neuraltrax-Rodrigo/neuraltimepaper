**# Tools**




\- Nome: Branding Guidelines

  Tipo: Integração

  Função: Acessar diretrizes visuais da marca (cores, tipografia, estilo, elementos)

  Permissões:

  - Leitura de guidelines visuais completos

  - Validação de peças contra identidade visual

  - Acesso a exemplos de material aprovado




\- Nome: Copywriter

  Tipo: Integração

  Função: Receber copy e alinhar conceito visual com mensagem

  Permissões:

  - Leitura de copies de campanhas

  - Coordenação visual + copy

  - Validação de hierarquia texto-imagem




\- Nome: Agente Alfredo Soares

  Tipo: Integração

  Função: Receber estratégia de marketing e posicionamento de campanha

  Permissões:

  - Leitura de briefing estratégico

  - Acesso a dados de público e persona

  - Insights de diferenciais e ângulos de campanha




\- Nome: Agente de KPIs

  Tipo: Integração

  Função: Receber dados de performance visual de peças anteriores

  Permissões:

  - Leitura de CTR por tipo de visual

  - Dados de engajamento por estilo visual

  - Resultados de A/B tests visuais




\- Nome: Visual Trend Board

  Tipo: API

  Função: Pesquisar tendências visuais e referências de design

  Permissões:

  - Buscar referências visuais por estilo e setor

  - Acessar tendências de design contemporâneo

  - Curadoria de mood boards e inspirações




\- Nome: Composition Analyzer

  Tipo: Script

  Função: Analisar composição visual de peças (hierarquia, contraste, balanço, espaço)

  Permissões:

  - Avaliar hierarquia visual

  - Medir contraste e legibilidade

  - Verificar regra dos terços e proporção áurea

  - Analisar distribuição de elementos




\- Nome: Image Curator Tool

  Tipo: API

  Função: Buscar e curar imagens de bancos (Unsplash, Shutterstock, Pexels)

  Permissões:

  - Buscar por estilo, cor, conceito

  - Filtrar por qualidade e licença

  - Organizar mood boards e referências

  - Sugerir alternativas visuais




\- Nome: Color Palette Generator

  Tipo: Script

  Função: Gerar e validar paletas de cores para campanhas

  Permissões:

  - Gerar paletas harmônicas baseadas em cor primária

  - Validar contraste para acessibilidade (WCAG)

  - Sugerir combinações por emoção/mensagem

  - Manter consistência com paleta da marca
