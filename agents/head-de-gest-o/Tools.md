# Tools

## Claude API (Anthropic)

* **Tipo:** API
* **Função:** Motor de síntese e raciocínio — processa os inputs dos clones, dados do Head de Dados e diretrizes do CEO para gerar recomendações de gestão integradas e contextualizadas
* **Permissões:**
  * Simular perspectivas dos 6 clones sobre um desafio de gestão
  * Sintetizar visões complementares em recomendação única e coerente
  * Redigir conselhos de gestão em linguagem executiva, prática e contextualizada para a Neural Trax
  * Analisar dados operacionais recebidos do Head de Dados e gerar interpretação comportamental/organizacional

## Slack MCP

* **Tipo:** Integração / MCP Server
* **Função:** Canal de recebimento de contexto (do CEO e Head de Dados) e entrega de recomendações de gestão
* **Permissões:**
  * Receber alertas e dados do Head de Dados via canal dedicado
  * Receber diretrizes e perguntas do CEO via DM ou canal executivo
  * Enviar recomendações de gestão ao CEO em formato estruturado
  * Consultar histórico de conversas para manter contexto entre sessões

## Google Sheets (via Zapier MCP)

* **Tipo:** Integração
* **Função:** Acesso ao dashboard de KPIs gerado pelo Head de Dados para contextualizar recomendações de gestão
* **Permissões:**
  * Ler KPIs de todas as frentes do dashboard central
  * Consultar histórico de métricas para identificar padrões comportamentais e organizacionais
  * Registrar recomendações entregues e status de implementação (rastreabilidade de conselhos)

## Asana MCP

* **Tipo:** Integração / MCP Server
* **Função:** Acompanhamento de OKRs e iniciativas para contextualizar recomendações de gestão no andamento real da operação
* **Permissões:**
  * Ler status de projetos e tarefas das lideranças
  * Verificar bloqueios e atrasos que demandam orientação de gestão
  * Registrar recomendações como tarefas acionáveis quando necessário
