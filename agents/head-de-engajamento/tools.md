# Tools

## Claude API (Anthropic)

* **Tipo:** API
* **Função:** Geração de copies, briefings criativos, análise de engajamento e síntese de relatórios do squad
* **Permissões:**
  * Redigir copies para posts, anúncios, landing pages e emails
  * Gerar briefings criativos para o squad com base no ICP e calendário
  * Sintetizar relatório semanal de engajamento para o CEO

## Slack MCP

* **Tipo:** Integração / MCP Server
* **Função:** Comunicação com o squad e reporte ao CEO
* **Permissões:**
  * Receber relatórios semanais de todos os agentes do squad
  * Enviar briefings, aprovações e feedbacks criativos
  * Publicar relatório semanal de engajamento ao CEO

## Google Sheets (via Zapier MCP)

* **Tipo:** Integração
* **Função:** Dashboard de engajamento — métricas de redes sociais, web e performance de conteúdo
* **Permissões:**
  * Consolidar métricas semanais de todos os canais (Instagram, LinkedIn, site, email)
  * Acompanhar calendário editorial e status de entregas criativas
  * Registrar performance de cada formato e conteúdo para análise histórica

## Asana MCP

* **Tipo:** Integração / MCP Server
* **Função:** Gestão do calendário editorial, projetos criativos e entregas do squad
* **Permissões:**
  * Criar e gerenciar tarefas criativas com prazo e responsável
  * Acompanhar status de produção de conteúdo, LPs e peças de design
  * Bloquear entregas sem aprovação do Diretor de Arte ou Head de Engajamento

## Zapier MCP

* **Tipo:** Integração / MCP Server
* **Função:** Automações de publicação e coleta de dados de engajamento das plataformas
* **Permissões:**
  * Sincronizar métricas de Instagram e LinkedIn com o Google Sheets
  * Disparar alertas quando engajamento de post cair abaixo do benchmark
  * Automatizar publicações programadas do calendário editorial
