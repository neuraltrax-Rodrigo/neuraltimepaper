# User

## CEO (Rodrigo)

* **O que espera:** Marca consistente, presença digital crescente e conteúdo que atrai o ICP — sem precisar aprovar cada peça, mas com visibilidade sobre o que está sendo publicado e como está performando
* **Quando acionar diretamente:** Mudança de identidade visual, novo posicionamento ou campanha de grande impacto

***

## Squad (fontes de dados e entregas)

| Agente              | O que reporta / entrega                                                       |
| ------------------- | ----------------------------------------------------------------------------- |
| Branding 🔴         | Inconsistências de marca identificadas, atualizações do brandbook             |
| Copywriter          | Copies entregues, testes A/B ativos, resultados de copy testada               |
| Diretor de Arte     | Peças aprovadas na semana, direções criativas em andamento                    |
| Criador de Conteúdo | Conteúdo publicado, formatos testados, performance por peça                   |
| UX Designer         | Wireframes e fluxos entregues, colaborações com PM e Web Designer             |
| Paulo Cuenca        | Alcance autoral, seguidores qualificados, engajamento por post                |
| Social Media        | Alcance, impressões, engajamento, seguidores novos, melhor conteúdo da semana |
| Web Designer 🔴     | Status de páginas em produção, LPs entregues, bugs de site                    |
| Analista Web        | Sessões, bounce rate, conversões de página, recomendações de CRO              |

***

## Head de Tráfego (parceiro de conteúdo)

* **Relação:** Consome criativos produzidos pelo squad para campanhas pagas. Fornece feedback de performance de criativo (CTR, CPL por peça) que alimenta as decisões do Head de Engajamento sobre o que funciona para o ICP.

***

## Head de Conversão (parceiro de LP)

* **Relação:** Solicita landing pages para campanhas e fornece dados de conversão de LP que orientam otimizações do Web Designer e Criador de Landing Page.
