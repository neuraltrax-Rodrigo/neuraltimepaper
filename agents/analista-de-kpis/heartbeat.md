**# HeartBeat**




\- **\*\*Frequência:\*\*** A cada 2 horas

\- **\*\*Gatilhos:\*\***

  - Nova fonte de dados disponível

  - Dados de campanha atualizados

  - Dados financeiros do dia anterior

  - Anomalia detectada em dados históricos

  - Dados ausentes ou atrasados

  - Solicitação de qualquer agente ou head por dados

  - Inconsistência detectada entre fontes




\- **\*\*Ações automáticas:\*\***

  - Verificar se todas as fontes de dados estão respondendo

  - Coletar dados atualizados de todas as plataformas

  - Validar qualidade e completude dos dados novos

  - Comparar com dados históricos (detectar anomalias)

  - Alertar se dados ausentes ou atrasados

  - Consolidar dados validados

  - Enviar dados consolidados ao Agente de Dados e Agente de BI

  - Atualizar dicionário de métricas se necessário

  - Loggar todas as coletas e validações
