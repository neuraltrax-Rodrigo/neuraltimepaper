# CronJobs

- **Nome:** Weekly Usability Check
  **Frequência:**
    - 0 9 * * 1 (Toda segunda-feira às 9:00)
  
  **Trigger:**
    - Início da semana de trabalho
  
  **Agent:**
    - UX Designer Agent
  
  **Skill:**
    - Usability Testing
  
  **Input:**
    - Protótipos em desenvolvimento
    - Tarefas críticas definidas pela equipe
  
  **Ação:**
    - Agendar sessões de teste de usabilidade semanais
    - Recrutar 5 participantes para testes rápidos
    - Compilar resultados preliminares
  
  **Output:**
    - Relatório semanal de usabilidade
    - Lista priorizada de melhorias para a sprint

- **Nome:** Daily Analytics Review
  **Frequência:**
    - 0 8 * * * (Todos os dias às 8:00)
  
  **Trigger:**
    - Disponibilidade de novos dados de analytics
  
  **Agent:**
    - UX Designer Agent
  
  **Skill:**
    - User Research Analysis
  
  **Input:**
    - Métricas de engajamento do dia anterior
    - Funnel de conversão
    - Heatmaps e gravações de sessão
  
  **Ação:**
    - Coletar e analisar métricas de uso
    - Identificar anomalias ou oportunidades
    - Gerar insights para otimização
  
  **Output:**
    - Dashboard atualizado de métricas UX
    - Alertas sobre quedas de conversão ou problemas

- **Nome:** Monthly Accessibility Audit
  **Frequência:**
    - 0 10 1 * * (Primeiro dia de cada mês às 10:00)
  
  **Trigger:**
    - Início do mês calendário
  
  **Agent:**
    - UX Designer Agent
  
  **Skill:**
    - Accessibility Audit
  
  **Input:**
    - Interfaces produzidas no mês anterior
    - Checklist WCAG 2.1 AA
  
  **Ação:**
    - Executar auditoria completa de acessibilidade
    - Testar com ferramentas assistivas
    - Documentar violações encontradas
  
  **Output:**
    - Relatório mensal de conformidade
    - Backlog de correções de acessibilidade
    - Certificação de conformidade (se aplicável)
