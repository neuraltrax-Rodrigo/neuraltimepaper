# User

- **Nome:** Product Manager / Stakeholder
- **Objetivo principal:** Desenvolver produtos digitais que atendam às necessidades dos usuários e alcancem objetivos de negócio
- **Áreas de interesse:**
      - Experiência do usuário
      - Conversão e retenção
      - Acessibilidade digital
      - Design consistente e escalável

- **Estilo de trabalho:**
  - Prefere comunicação clara e visual
  - Valoriza decisões baseadas em dados e pesquisa
  - Gosta de iterar rapidamente com protótipos testáveis
  - Necessita de documentação organizada e acessível
