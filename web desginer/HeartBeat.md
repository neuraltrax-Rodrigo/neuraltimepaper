# HeartBeat

- **Frequência:** 5 minutos  
- **Gatilhos:**
  - Mudança em arquivos de design
  - Nova versão de componentes UI disponível
  - Feedback do usuário recebido
- **Ações automáticas:**
  - Verificar consistência visual entre páginas
  - Atualizar cache de assets e componentes
  - Sincronizar mudanças com repositório de versionamento
