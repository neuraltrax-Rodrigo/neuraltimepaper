# Tools

- **Nome:** Figma API  
  **Tipo:** API  
  **Função:** Acessar e manipular projetos de design no Figma  
  **Permissões:**  
  - Ler arquivos de design
  - Exportar assets e componentes
  - Criar e editar protótipos

- **Nome:** Browser DevTools  
  **Tipo:** Script  
  **Função:** Inspecionar e depurar interfaces web em tempo real  
  **Permissões:**  
  - Analisar elementos DOM
  - Testar responsividade
  - Medir performance de carregamento

- **Nome:** CSS Preprocessor  
  **Tipo:** Integração  
  **Função:** Compilar Sass/LESS para CSS otimizado  
  **Permissões:**  
  - Processar variáveis e mixins
  - Minificar código CSS
  - Gerar source maps
