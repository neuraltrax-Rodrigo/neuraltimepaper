# Skills

- **Nome:** ResponsiveLayoutBuilder  
  **Descrição:** Cria layouts responsivos adaptáveis a diferentes dispositivos e tamanhos de tela  
  **Input:**
    - Estrutura HTML base
    - Breakpoints desejados
    - Diretrizes de design
  **Output:**
    - Código CSS responsivo completo
    - Preview do layout em diferentes resoluções
  **Tools:**
    - CSS Preprocessor
    - Browser DevTools
  **Processo:**
    1. Analisar estrutura HTML e requisitos de breakpoints
    2. Gerar media queries otimizadas para cada dispositivo
    3. Testar e ajustar layout em múltiplas resoluções
  **Regras:**
    - Mobile-first como abordagem padrão
    - Manter performance de carregamento abaixo de 3 segundos

- **Nome:** AccessibilityAuditor  
  **Descrição:** Avalia e corrige problemas de acessibilidade em interfaces web  
  **Input:**
    - URL ou código HTML da página
    - Nível de conformidade WCAG desejado (A, AA, AAA)
  **Output:**
    - Relatório de problemas de acessibilidade
    - Sugestões de correção priorizadas
  **Tools:**
    - Browser DevTools
  **Processo:**
    1. Escanear página em busca de violações de acessibilidade
    2. Classificar problemas por severidade e impacto
    3. Gerar recomendações específicas de correção
  **Regras:**
    - Priorizar correções que impactam usuários com deficiência
    - Garantir conformidade mínima com WCAG 2.1 Level AA
